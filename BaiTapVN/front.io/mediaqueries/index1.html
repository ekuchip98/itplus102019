Cú pháp của media query :
        @media screen and (condition) {
            selector1 {
                // code css cho selector1
            }
        }

        Kỹ thuật media query dùng để sử lý cho các màn hình có kích cỡ khác nhau
        như là smartphone , máy tính bảng ipad , laptop , màn hình desktop
