Đã học 3 giá trị of thuộc tính display
        display: block
        display: inline
        display: none
        => Giá trị kết hợp giữa block & inline: inline-block
        display: inline-block

        ******** CẦN GHI NHỚ *********
        đặt css width or height cho thẻ có kiểu display là inline không có tác dụng vd: a,span
        đặt css witd or height cho thẻ có kiểu display là block có tác dụng vd:div h1-6
